Equipamentos Necessários

EQUIPAMENTOS PARA TRABALHO HOME OFFICE

Para iniciar suas atividades, a Vivo fornecerá todos os equipamentos necessários e internet de 50 Mb

Computador ou Notebook

Requisitos mínimos: Processador Intel Core i3 (ou similar), 4GB RAM, 128GB de armazenamento

Recomendado: Processador Intel Core i5 (ou similar), 8GB RAM, 256GB de armazenamento

Conexão de Internet

Requisitos mínimos: Velocidade de 15 Mbps para download e 5 Mbps para upload

Recomendado: Velocidade de 50 Mbps para download e 10 Mbps para upload

Headset com Microfone

Requisitos mínimos: Headset com microfone integrado e conexão USB ou P2

Recomendado: Headset com cancelamento de ruído e microfone direcional

A VIVO fornece todos os equipamentos necessários que são obrigatórios para o trabalho, mas é preciso pagar uma Taxa de garantia no valor de R$ 47,90 para receber os equipamentos da Vivo direto na sua casa.

IMPORTANTE: Os equipamentos de trabalho (Notebook, Colete e HeadSet) são enviados com garantia de devolução. Para segurança, é cobrada uma taxa de R$ 47,90, que será estornada no seu primeiro pagamento. Essa medida evita fraudes, como candidatos que recebem os equipamentos e não seguem com o trabalho. A devolução dos itens é obrigatória em caso de desligamento.
